# 晓园 Vault 产品定义文档

> 创建时间：2026-04-23
> 更新时间：2026-04-24
> 目标：5-10 人团队/部门的知识库管理工具
> 形态：Electron 桌面应用 + 移动端 App（Capacitor）

---

## 一、产品定位

**一句话：** AI 增强版 Obsidian，面向中小团队

**核心价值：**
- 像 Obsidian 一样本地优先、文件透明
- 内置 AI 自动分类、自动标签、知识图谱
- 混合搜索（语义 + 关键词）
- 移动端支持（iOS/Android）
- 云端同步（Phase 2）

---

## 二、目标客户

| 维度 | 定义 |
|------|------|
| **规模** | 5-10 人团队/部门 |
| **场景** | 项目知识沉淀、部门文档管理、小型智库 |

## 四、数据架构

### 真相源：本地 Markdown 文件

```
MyVault/
├── 01-项目管理/
│   ├── 项目A.md              # Markdown + YAML frontmatter
│   └── 附件.pdf
├── 02-知识库/
│   └── 合成生物学.md
├── attachments/              # 附件存放
└── .xiaoyuan/
    ├── index.sqlite          # 本地索引（AI搜索用）
    ├── config.json           # 配置
    └── cache/                # AI 缓存
```

**关键设计：**
- `.md` 文件是真相源，随时可用 Obsidian、Typora、VS Code 打开
- `.xiaoyuan/` 是应用数据，删除后 AI 功能失效，Markdown 文件不受影响

### 同步方案（Phase 2）

```
用户自己选择同步方式（我们推荐，不强制绑定）：
├── 坚果云团队版（¥199/年）
├── Syncthing（免费，P2P）
├── OneDrive（微软生态）
├── iCloud（苹果全家桶）
└── 群晖 NAS（自建）

Phase 2：提供可选的云端 Sync 服务（阿里云 VPS）
```

## 五、功能边界

### MVP（Phase 1，单机版）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| **Markdown 编辑器** | 所见即所得，支持 YAML frontmatter | P0 |
| **文件夹树** | 左侧导航，支持拖拽 | P0 |
| **AI 分类** | 自动推荐文件夹 | P0 |
| **AI 标签** | 自动提取关键词标签 | P0 |
| **本地全文搜索** | SQLite FTS5 关键词搜索 | P0 |
| **AI 摘要** | 长文档生成摘要 | P0 |
| **AI 推理** | RAG 问答，基于 Vault 内容回答 | P0 |
| **AI 写作** | 输入大纲生成完整文档 | P0 |
| **AI 语义搜索** | 语义理解 + 关键词混合搜索 | P1 |
| **知识图谱** | 基于标签/链接的 D3.js 图谱 | P1 |
| **raw files 文件夹** | 原始文件保留 | P1 |
| **文件导入** | PDF/Word/图片 → Markdown | P2 |

### Phase 2（移动端 + 云端 Sync）

| 功能 | 说明 |
|------|------|
| **iOS/Android App** | Capacitor 打包，移动端查看 + 搜索 |
| **云端 Sync** | 阿里云 VPS，轻量级 sync 服务 |
| **端到端加密** | 我们服务器只存密文 |
| **冲突处理** | Last-Write-Wins + 冲突副本 |

### 不做（当前阶段）

| 功能 | 原因 |
|------|------|
| 文档级权限 | 5-10 人用不上，Vault 级隔离够用 |
| 版本历史 | 同步服务自带（坚果云/Syncthing） |
| 评论/@mention | 后期加 |
| 插件系统 | 验证 PMF 后再做 |
| 自建 AI 推理 | Phase 2 验证后评估 |
